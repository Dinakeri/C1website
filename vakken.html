Verschillende vakken

  1. Web Development: Met Web bedoelen we natuurlijk het bouwen van websites, als eerstejaars zal de focus hier vooral liggen op HTML, CSS en Javascript
  2. Native Development: Terwijl web meer focust op het website gedeelte en het opmaak van alles is Native meer de praktische kant. Je leert echt code schrijven en zal een begin hieraan maken met Python
  3. Praktijklessen: Hierin zul je de kennis van zowel Web als Native toepassen door opdrachten te krijgen waar je alleen of met een team aan zal werken. Dit is een soort test of je alle kennis goed kent en kunt gebruiken.
  4. Praktijk Ondersteuning: Natuurlijk om te coderen heb je bepaalde tools nodig die je hierbij helpen, en dit vak leer je daarover, tools zoals Git.
  5. Generieke vakken: De vakken die horen tot elke opleiding voor het centrale examen, dus: Engels, Nederlands en Rekenen
  6. Burgerschap: Bij Burgerschap leer je argumenteren en kritisch denken
  7. Algoritme: Hierbij leer je hoe Algoritmes werken
  8. Digitale Vaardigheden: Digitale Vaardigheden leer je surface-level kennis van wat je moet weten, dus het navigeren van je laptop of het maken van een Spreadsheet
